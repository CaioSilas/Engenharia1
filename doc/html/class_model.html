Detailed Description

Classe responsável por representar um modelo.

Um modelo contém sistemas e fluxos que são usados para simular um sistema real ou teórico. Exemplo de uso:

System *p1= new System("p1", 100.0);
System *p2= new System("p2", 10.0);
FlowLogistico* logFlow = new FlowLogistico("logFlow", p1,p2);
Model *logModel= new Model("logModel",0,100,1);
logModel->add(p1);
logModel->add(p2);
logModel->add(logFlow);
logModel->execute();
delete(logModel);
delete(p1);
delete(p2);
delete(logFlow);
